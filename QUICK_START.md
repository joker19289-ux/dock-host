# BoyJack OS v1.0 — Quick Start

## Access Right Now

**Local Machine:**
- Web: http://localhost:5000
- VNC: localhost:5901

**Remote (from another machine):**
- Web: http://<IP>:5000/vnc.html
- VNC: <IP>:5901

## First Time Setup

1. **Click the "BJ Start" button** on the taskbar (bottom-left)
2. **Select an application** from the menu
3. **Use Alt+F2 or Alt+F3** for quick access to Terminal/Files

## Essential Commands

```
Alt+F2         Open Terminal
Alt+F3         Open File Manager
Ctrl+F1-F4     Switch Workspaces
Alt+Tab        Switch Windows
Right-click    Desktop menu
```

## Applications

| App | Shortcut | Purpose |
|-----|----------|---------|
| Terminal | Alt+F2 | Command-line work |
| Files | Alt+F3 | Browse folders |
| Firefox | Menu | Web browsing |
| Editor | Menu | Edit files |

## Troubleshooting

**Not working?**
1. Refresh browser (Ctrl+F5)
2. Check URL: http://localhost:5000
3. Wait 15 seconds for boot to complete
4. See USER_GUIDE.md troubleshooting section

**Need help?**
- Check logs: `/tmp/fluxbox.log`
- Read documentation files
- Check browser console (F12)
